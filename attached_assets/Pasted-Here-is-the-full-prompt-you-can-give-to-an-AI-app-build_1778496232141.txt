MAIN PRINCIPLES==================================================The app should be:- offline- private- fast- minimal- convenient- smooth- easy to use with one hand- not overloaded- focused on daily life and study- built with efficient data structures- easy to backup and transferThe app should not become a generic todo app.It should feel like a personal daily reset system plus a clean study tracker.The main user problem:I forget small daily things, temporary tasks, and want to track study time by subject using Pomodoro.
Temporary tasks are also called variable tasks: tasks added for today only, not part of the default daily routine, and they disappear after the day ends unless the user manually moves them to tomorrow.

Every task card must have a very clear one-tap checkbox/tick/mark button that changes the task status to “Done!” immediately. This should be the fastest action in the app.

Adding a task should require as few taps as possible. The app should support quick-add from the Routine screen and quick-start from the Study subject cards.
The main solution:A simple offline app where routine tasks reset daily, temporary tasks disappear after a day, and study tracking is separate with subject-based Pomodoro statistics.
